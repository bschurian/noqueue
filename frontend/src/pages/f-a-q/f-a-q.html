FAQ

How can I line up for multiple queues?

At this point, it is not possible to enter multiple queues.

I created a shop but I can't find the clients queue.

You need to add yourself as an employee first in order to be able to treat clients.

I created a shop but can't find when searching.

For user experience reasons, only shops with active services are shown as search results.

I accidentally deleted my shop. Can I restore it back?

Unfortunately, this functionality is not supported in this version of NoQueue.

I'm not delighted with the treatment in a shop that I queued in for with NoQueue.

NoQueue doesn't hold any warranty for the quality of client treatment. Later versions of the application may include a rating-system of the shops.

I lined up for a shop that doesn't exist.

When saving a new shop, we only check if its' address exists, but not the shop itself.
